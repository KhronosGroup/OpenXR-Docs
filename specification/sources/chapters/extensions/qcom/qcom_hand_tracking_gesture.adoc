==== Overview

This extension enables applications to detect hand gestures.

==== Get gesture detection data

[open,refpage='xrGetHandGestureQCOM',type='protos',desc='Gets gesture detection data',xrefs='']
--
The flink:xrGetHandGestureQCOM function is defined as:

include::{generated}/api/protos/xrGetHandGestureQCOM.adoc[]

.Parameter Descriptions
****
* pname:handTracker is an slink:XrHandTrackerEXT previously created by
  flink:xrCreateHandTrackerEXT.
* pname:time is the basetype:XrTime of gesture detection.
* pname:handGesture is a pointer to slink:XrHandGestureQCOM and returns the
  current gesture detection data.
****

The flink:xrGetHandGestureQCOM function retrieves the hand gesture data at a
given time.

[open,refpage='XrHandGestureQCOM',type='structs',desc='Gesture detection data',xrefs='']
--
The slink:XrHandGestureQCOM structure is defined as:

include::{generated}/api/structs/XrHandGestureQCOM.adoc[]

.Member Descriptions
****
* pname:gesture is the elink:XrHandGestureTypeQCOM detected.
* pname:gestureRatio is a value between 0 and 1 indicating how much the
  gesture is applied.
* pname:flipRatio is a value between -1 and 1 indicating if the hand gesture
  is detected from the back, from the front, or in between.
  The value is 1 when the palm is facing the user and is -1 when the palm is
  directed toward the environment.
  It represents the normalized dot product between the palm normal and the
  palm-to-user vector.
****

slink:XrHandGestureQCOM structure returns the state of the gesture
detection.
