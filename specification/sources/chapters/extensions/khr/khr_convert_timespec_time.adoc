*Overview*

This extension provides two functions for converting between timespec
monotonic time and basetype:XrTime.
The flink:xrConvertTimespecTimeToTimeKHR function converts from timespec
time to basetype:XrTime, while the flink:xrConvertTimeToTimespecTimeKHR
function converts basetype:XrTime to timespec monotonic time.
The primary use case for this functionality is to be able to synchronize
events between the local system and the OpenXR system.

include::../../../../generated/api/protos/xrConvertTimespecTimeToTimeKHR.txt[]

.Parameter Descriptions
****
* pname:instance is an slink:XrInstance handle previously created with
  flink:xrCreateInstance.
* pname:timespecTime is a code:timespec obtained from code:clock_gettime with
  code:CLOCK_MONOTONIC.
* pname:time is the resulting basetype:XrTime that is equivalent to the
  pname:timespecTime.
****

The flink:xrConvertTimespecTimeToTimeKHR function converts a time obtained
by the code:clock_gettime function to the equivalent basetype:XrTime.

If the output pname:time cannot represent the input pname:timespecTime, the
runtime must: return ename:XR_ERROR_TIME_INVALID.

.Parameter Descriptions
****
* pname:instance is an slink:XrInstance handle previously created with
  flink:xrCreateInstance.
* pname:time is an basetype:XrTime.
* pname:timespecTime is the resulting timespec time that is equivalent to a
  code:timespec obtained from code:clock_gettime with code:CLOCK_MONOTONIC.
****

The flink:xrConvertTimeToTimespecTimeKHR function converts an
basetype:XrTime to time as if generated by code:clock_gettime.

If the output pname:timespecTime cannot represent the input pname:time, the
runtime must: return ename:XR_ERROR_TIME_INVALID.
